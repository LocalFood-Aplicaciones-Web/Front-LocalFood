## üöÄ Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the JSON Server (API)**
   ```bash
   npm run api
   ```
   The API will run on `http://localhost:3000`

3. **Start the Development Server** (in a new terminal)
   ```bash
   npm run dev
   ```
   The app will run on `http://localhost:5173`
